## 说明

- `T1`：`A -> B` 停车
- `T2`：顺时针一圈 `A -> B -> C -> D -> A`
- `T3`：逆时针一圈 `A -> D -> C -> B -> A`

- 单击：启动 / 停车
- 停车时双击：切换任务模式
- 停车时长按：重置当前任务状态
