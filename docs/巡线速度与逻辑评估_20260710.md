## 当前高速三档版逻辑

1. 每 `5 ms` 读一次 8 路灰度。
2. `black_count >= 7` 连续 2 次，判断为十字/大面积黑线，进入约 `160 ms` 直冲：
   - `Move_X = 310 mm/s`
   - `Move_Z = 0`
   - 之后约 `300 ms` 冷却，防止十字区域重复触发。
3. 正常巡线按偏差分段速度：
   - `abs(pos) < 10 mm`：`300 mm/s`
   - `10 mm <= abs(pos) < 22 mm`：`240 mm/s`
   - `abs(pos) >= 22 mm` 或 `black_count >= 5`：`185 mm/s`
4. 全白丢线：
   - `Move_X = 0`
   - 按上一次线偏移方向以 `1.80 rad/s` 原地找线。
5. 转向：
   - `GRAY_STEER_GAIN = 1.70`
   - `GRAY_MAX_ANGULAR_SPEED = 3.60`

## 实车测试建议

1. 先用当前高速三档版跑一圈，重点观察：
   - 直线是否开始左右摆动。
   - 弯道是否压外侧出线。
   - 十字路口是否直冲过头。
   - 电池电压下降后速度是否明显变化。
2. 如果直线摆动：
   - 先把 `GRAY_STEER_GAIN` 从 `1.70` 降到 `1.55`。
   - 或把直线速度 `300` 降到 `280`。
3. 如果弯道冲出：
   - 先把 `GRAY_CURVE_SPEED_MM_S` 从 `185` 降到 `165`。
   - 或把 `GRAY_CURVE_POS_MM` 从 `22` 降到 `18`，让它更早降速。
4. 如果十字冲过头：
   - 把 `GRAY_CROSS_RUN_TICKS` 从 `32` 降到 `24`。
5. 如果十字没冲过去：
   - 把 `GRAY_CROSS_RUN_TICKS` 从 `32` 提到 `38`。
6. 如果全白找线方向反了：
   - 交换 `last_search_dir = -1` 和 `last_search_dir = 1` 两处赋值。

## 推荐回退顺序

1. 当前高速三档版：用于冲速度。
2. 简单高速版：如果当前版太激进，回退到直线 `220` / 弯道 `160`。
3. 2.3倍保底版：如果地图局部还有小半径或灰度不稳定，回退到基准 `105`。
4. 原始低速稳定版：只用于排查硬件、线序、电机方向。
