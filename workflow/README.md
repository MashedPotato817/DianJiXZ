## 视觉协议帧（草案）

```
帧头    目标X    目标Y    面积    校验
0xAA   [2B]     [2B]     [2B]    [1B]
```

- 坐标范围：0~320（X）、0~240（Y）（QVGA 分辨率）
- 面积：色块像素数，未检测到目标时填 0
- 校验：前 7 字节异或

### MSPM0 侧解析伪代码

```c
if (data == 0xAA) {
    index = 0;
}
buf[index++] = data;
if (index >= 8) {
    if (xor_check_ok(buf)) {
        target_x = buf[1] | (buf[2] << 8);
        target_y = buf[3] | (buf[4] << 8);
        target_area = buf[5] | (buf[6] << 8);
    }
    index = 0;
}
```
