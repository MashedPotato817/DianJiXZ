# 2026 电赛赛题猜测与备赛路线

> 本文是赛前技术预案，不是官方赛题信息，也不对题目方向作保证。文中的器件组合来自赛前线索整理；原始“26 年电赛材料清单”未随仓库提交，因此结论应按“可复用的备赛投资”理解，而非材料清单的逐条复述。

## 1. 结论先行

最值得投入的主线是：**可调倾角管道中的钢球运动测量与闭环控制装置**。即使最终题目不是这一原型，完成该装置仍会沉淀光电计时、电磁执行、传感器标定、位置/速度估计和闭环控制等控制类通用能力。

建议按下面的顺序推进：

```text
光电测量与可靠计时
    > 电磁铁可靠释放
    > 倾角执行与角度闭环
    > 钢球位置估计与 PID
    > K230 视觉
    > 无线通信与展示界面
```

## 2. 预测方法与概率含义

### 2.1 使用的证据

本预测只采用三类证据：

1. **器件组合线索**：钢球、PPR 管、电磁器件、光电/角度传感器、执行电机、摄像头与显示通信模块能否形成一个闭合的任务系统。
2. **几何与工程可实现性**：4 分 PPR 管外径约 20 mm、壁厚约 3.4 mm，内径约为 \(20-2\times3.4=13.2\) mm；直径约 10 mm 的钢球可在其中滚动。这种尺寸匹配对“管内钢球”解释力很强。
3. **历年同类题目**：本目录的历年题库中已有 2017 M“管道内钢珠运动测量装置”、2017 B“滚球控制系统”、2015 J“小球滚动控制系统”、2013 J“电磁控制运动装置”和 2019 H/J“模拟电磁曲射炮”。它们说明相关测量、控制和电磁执行模块都曾作为独立考点出现。

### 2.2 概率不是统计概率

下表的百分比是把多个候选方向按解释力归一化后的**主观优先级**：它回答的是“备赛时间先投在哪里”，不表示官方出题概率，更不能简单相加后用于下注。每个方向还给出反证条件，赛前一旦获得更可靠的正式信息，应立即重排优先级。

| 候选方向 | 备赛权重 | 为什么排在这里 | 会降低权重的信号 |
| --- | ---: | --- | --- |
| 管道内钢球运动测量与控制 | 45% | 同时解释钢球、PPR 管、光电、角度、电磁铁与电机，且与 2017 M 高度接近 | 明确给出平面靶区、发射距离或小车场地，而不需要管道 |
| 滚球位置闭环控制 | 25% | 能解释钢球、位置检测、角度传感器和电机，控制难度与电赛风格匹配 | 题目只要求通过计时/测量，没有位置稳定或轨迹要求 |
| 电磁钢球发射或牵引 | 20% | 电磁铁、漆包线、磁芯和钢球构成完整执行链，也有历史参照 | 未给出靶区、距离、落点或重复发射等任务线索 |
| 小车寻找并搬运钢球 | 10% | 可利用摄像头、电磁铁和无线模块，但无法自然解释指定尺寸的管材 | 出现循迹地图、装卸区、车体尺寸和路径规则 |

## 3. 候选方向的详细拆解

### 3.1 方向 A：管道内钢球运动测量与控制（主线）

**核心猜测**：钢球由电磁铁保持或释放，在可调倾角的透明管内运动；系统要测量其运动过程，并可能进一步把到达时间、末端位置或速度控制到目标范围。

**为什么它最可信**：

- PPR 管与 1 cm 钢球的尺寸配合提供了明确的机械约束；若只是做小车或电磁发射，指定管径的必要性较弱。
- 光电传感器最适合构造两个已知间距的测速门；角度传感器和电机又自然对应“可调坡度”。
- 电磁铁可以做重复性更好的起始条件：吸附、等待、释放，比人工放球更易计时和评分。
- 2017 M 已验证“管道 + 钢珠 + 运动测量”可以独立成题；本方向是在其基础上加入执行和闭环的合理升级，而非凭空假设。

**可能的任务层级**：

| 层级 | 可能任务 | 最小实现 |
| --- | --- | --- |
| 基础 | 释放钢球，测量两点通过时间，显示平均速度 | 两个光电门 + 定时器捕获 + OLED |
| 发挥 | 记录多段时间，估计速度/加速度，比较多次一致性 | 三个光电门 + 时间戳数组 + 标定距离 |
| 提高 | 调节倾角，使钢球在给定时间到终点或速度落在范围内 | 倾角执行器 + 角度反馈 + 迭代调角 |
| 扩展 | 连续估计位置，停在目标区或跟踪给定轨迹 | 相机/线性传感器 + 位置环 + 保护逻辑 |

**建议的系统链路**：

```text
按键/串口启动 → 电磁铁释放 → 光电门或相机采样
        ↓                          ↓
  释放状态确认              速度/位置估计与滤波
        ↓                          ↓
        └───── 目标误差计算 ←──────┘
                       ↓
            倾角控制器 → 电机/舵机 → 管道角度
```

**真正要验收的风险**：先检查光电门是否会被钢球漏检、释放是否有粘连、管道是否会让球卡滞；这三项不稳定时，PID 和视觉都没有意义。

### 3.2 方向 B：滚球位置闭环控制（控制能力主线）

**核心猜测**：在管道、横梁或平板上，通过改变倾角，使钢球进入并保持在目标位置。它既可能是 A 的提高部分，也可能单独成题。

**为什么仍值得做**：2015 J、2017 B 与 2011 B 都说明“可观测位置 + 可调姿态 + 稳定控制”是持续出现的控制范式。即使最终载体不是钢球，角度内环、位置外环、限幅和滤波都可以复用到小车、云台或摆杆任务。

**控制分层**：

```text
外环：e_x = x_ref - x → 目标倾角 theta_ref
内环：e_theta = theta_ref - theta → 电机 PWM/舵机命令
```

- 初版可直接用单环 PID：位置误差直接映射为舵机目标角度，快速验证机构方向是否正确。
- 稳定版建议拆为双环：内环保证倾角快速、可预测，外环只处理较慢的位置动态，能显著减少机械迟滞带来的振荡。
- 位置测量优先级为：固定光电阵列（离散、鲁棒）→ 线性电位器/编码器（若结构允许）→ K230 相机（连续、调试成本高）。
- 必须加入死区、输出限幅、积分限幅和丢失目标保护；这些比盲目提高 PID 增益更重要。

**成功判据建议**：记录稳态误差、超调量、稳定时间和连续 10 次重复试验的成功率。单次“看起来能停住”不能代表装置可参赛。

### 3.3 方向 C：电磁钢球发射或牵引（可复用执行支线）

**核心猜测**：线圈/电磁铁通过可控电流和通电时间驱动钢球，到达指定区域或被相机识别落点。

**为什么不是第一优先级**：电磁器件确实能支持该方向，但 PPR 管、光电门和角度传感器对“纯发射”并非必要。它更像 A 的释放机构或一项可能的独立发挥题，应先完成安全、可控的低能量实验，再决定是否投入落点闭环。

**最小可行链路**：

```text
MCU 定时输出 → 栅极驱动/MOSFET → 线圈/电磁铁
                      ↓
              电流采样与过流保护
```

**硬件边界**：感性负载必须考虑关断瞬态。保持/吸附类电磁铁可采用续流二极管；若追求快速退磁或脉冲响应，则需按器件耐压设计 TVS、吸收网络或主动钳位，并先用限流电源验证温升。不要在未测线圈电阻、电流和温升前提高占空比或脉冲能量。

### 3.4 方向 D：小车寻找并搬运钢球（保留现有资产）

本仓库已有 MSPM0 差速车、巡线、K230 和串口通信基础，因此此方向的新增成本低。它的低权重只意味着不应以它作为 2026 预测的主项目，不代表应停止维护：可保留为视觉检测、电磁吸取与状态机的验证平台。

## 4. 备赛实施方案

### P0：先做出可测、可复现的底座

- [ ] 两个光电门检测钢球通过；用定时器输入捕获或微秒级时间戳计算速度。
- [ ] 电磁铁在连续 20 次试验中稳定吸附、释放，无明显残磁粘球。
- [ ] OLED/串口输出原始时间戳、测距、平均速度和错误码。
- [ ] 建立倾角执行机构，读取并校准角度；限定机械极限与软件极限。
- [ ] 完成 MOSFET 驱动、保险/限流、续流或钳位保护和温升检查。

### P1：把测量装置升级为控制装置

- [ ] 电机/舵机闭环到目标倾角，并测得阶跃响应。
- [ ] 完成位置 PID 的死区、限幅、积分限幅、手自动切换和急停。
- [ ] 完成钢球位置的低通或中值滤波，保留原始值便于调试。
- [ ] 定义统一串口帧：帧头、时间戳、位置/速度、状态字、校验。
- [ ] 完成零点标定、传感器超时、球丢失和执行器越界处理。

### P2：仅在底座稳定后投入

- [ ] K230 输出钢球横坐标、有效标志和置信度；固定背景优先灰度阈值、轮廓和圆检测，不先训练 YOLO。
- [ ] 自动搜索合适倾角或 PID 参数，并保存一次标定结果。
- [ ] 无线遥测、轨迹显示与上位机调参。
- [ ] 落点识别、电磁脉冲标定或更高阶状态反馈。

## 5. 最小验证装置与验收

```text
约 1 m 透明 PPR 管 + 直径约 1 cm 钢球
+ 两组光电传感器 + 一只电磁铁
+ 一个舵机或减速电机 + 一个角度传感器
+ MSPM0/STM32 + OLED/串口调试口
```

第一阶段只回答一个问题：**能否可靠、重复地测出钢球从 A 到 B 的时间？**

1. 固定两光电门距离，测量 10 次通过时间并保存结果。
2. 计算平均速度 \(v=d/\Delta t\)，对异常值给出明确错误码而非静默输出。
3. 改变同一倾角并重复试验，检查方差是否可接受。
4. 再加入电磁释放，比较人工释放与电磁释放的重复性。

通过后再进入位置闭环。这样能把“机构、测量、执行、控制”分开排错，避免一次性调整个系统。

## 7. 赛前决策规则

拿到正式赛题后，用 15 分钟完成以下判断：

1. 将题目中的每个指定器件映射到“测量、执行、结构、通信、显示”五类；无法解释的器件往往意味着漏掉了子任务。
2. 先圈出基础分所需的最短闭环：输入、测量、计算、输出、显示各至少一项。
3. 将 P0 模块直接拼成基础版本；只有基础版本连续稳定后才加入视觉、无线或自动调参。
4. 若正式题面与本文预测冲突，立即以题面为准；本文的价值在于缩短模块开发时间，而不是坚持猜测本身。
