01 / 几何约束

钢球 + PPR 管

约 10 mm 钢球可在约 13.2 mm 内径管中滚动，优先验证“管道内运动”而不是泛化的小车任务。

02 / 测量与执行

光电 + 角度 + 电机

天然构成测速、位置估计和可调坡度链路；先完成时间戳与角度校准，再做控制器。

03 / 历史参照

2017 M / B 等

证明管道钢珠测量与滚球闭环都曾是独立考点；可借鉴任务拆法，不能照搬题面。

从 P0 到 P2 的执行路径

每一步都有可独立验收的输出

P0-1

光电计时

两光电门、已知距离、时间戳和速度计算。

P0-2

电磁释放

连续 20 次释放，检查粘球、漏检和温升。

P0-3

倾角执行

建立电机/舵机和角度传感器闭环，限制机械极限。

P1

位置 PID

加入死区、限幅、积分保护与球丢失处理。

P2

视觉与展示

K230 坐标、无线遥测和自动调参；不阻塞基础功能。
