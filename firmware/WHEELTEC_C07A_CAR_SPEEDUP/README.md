# WHEELTEC C07A 巡线提速试验工程

本目录从已验证的 `WHEELTEC_C07A_CAR` 基础巡线版本独立复制，用于在不影响稳定版的前提下测试提速与弯道降速策略。对应开发分支为 `feat/line-follow-speedup`。

## 目标与当前策略

目标是在直线提高速度的同时，让小半径弯道主动降速，避免因横向偏差快速扩大而冲出黑线。

| 场景 | 条件 | 目标线速度 |
| --- | --- | --- |
| 直线/微偏 | 黑线偏差小于 `18 mm`，且黑线通道数少于 4 | `90 mm/s` |
| 弯道/宽黑线 | 偏差不小于 `18 mm`，或同时检测到至少 4 路黑线 | `75 mm/s` |
| 丢线搜线 | 沿最后有效偏线方向原地搜线 | `0.50 rad/s` 上限 |

直线速度是稳定版 `45 mm/s` 的两倍；弯道速度保守设置为 `75 mm/s`。常规巡线转向仍使用纯追踪差速控制，常规角速度上限保持 `0.40 rad/s`。

## 实车验证顺序

1. 抬车确认中线前进、左右偏线回线和丢线搜线方向均与稳定版一致。
2. 先跑直线，观察在 `90 mm/s` 下是否出现持续左右摆动。
3. 再跑小半径弯，确认传感器偏差增大时 OLED 上左右轮目标速度降至约 `75 mm/s`。
4. 若弯道仍冲出，优先降低 `GRAY_CURVE_SPEED_MM_S`；若回线反应不足，再小幅调整 `GRAY_STEER_GAIN`。一次只改一个参数。

## 关键文件

- `Control/control.h`：直线/弯道速度、阈值与搜线参数。
- `Control/control.c`：灰度质心、两档速度选择、差速控制与丢线搜线。
- `keil/empty_LP_MSPM0G3507_nortos_keil.uvprojx`：独立 Keil 工程，目标 `MSPM0G3507_Project`。
- `docs/PID与提速巡线审计_20260724.md`：当前代码快照、PID 调参边界与上下文交接信息。

使用 Keil 打开本目录的 `keil/empty_LP_MSPM0G3507_nortos_keil.uvprojx` 编译下载。SysConfig 文件仍须通过 TI SysConfig 修改和重新生成，不直接编辑 `ti_msp_dl_config.c/h`。
