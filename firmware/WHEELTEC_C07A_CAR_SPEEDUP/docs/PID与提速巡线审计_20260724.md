## 核心目标

调整左右轮速度 PI，使小车在连续运行时速度输出连贯、左右轮差小、横向摆动幅度小，并稳定沿黑线巡线。必须先分别确认“速度环是否稳定”和“外层巡线是否稳定”；不能把所有摆动都归因于 PID。

## 必须记住的事实

### 灰度与转向方向

- 物理左侧传感器是通道 0，物理右侧传感器是通道 7。
- 当前硬件定义为：黑线输出 `1`，白底输出 `0`；`GRAY_BLACK_LEVEL=1` 与实车一致。
- `Gray_Pos_mm` 从通道 0 的负值（左）递增到通道 7 的正值（右）。
- 黑线在右侧时，`Gray_Line_Pos_mm>0`，当前代码计算出 `Move_Z<0`；差速逆解使左轮更快、右轮更慢，车辆向右回线。左侧情况相反。
- 常规巡线是两轮同向前进的弧线转向，不是“一轮正转、另一轮反转”的原地转向。

### 电机、编码器与单位

- 目标轮速单位是 `m/s`；`Move_Z` 实际按 `rad/s` 使用。`Get_Target_Encoder()` 附近的“度/s”旧注释是错误的，不能据此改单位。
- 差速逆解：`V_left = Vx - Vz*Wheelspacing/2`，`V_right = Vx + Vz*Wheelspacing/2`。
- 速度测量时左编码器保持原符号，右编码器额外取反：`Encoder_B_pr = -OriginalEncoder.B`。
- 最终电机输出为 `Set_PWM(-MotorA.Motor_Pwm, -MotorB.Motor_Pwm)`；这是已按实车接线确认的方向适配。不要恢复成旧资料中的“只取反右轮”。

### 丢线状态

- 首次上电全白：停车，不能自行搜线。
- 已经识别过偏线后全白：停止前进，沿最后一次有效偏线方向低速原地搜线。
- 搜线方向只在 `Move_Z != 0` 的有效灰度帧更新；居中直线不会清除上一次有效方向。
- 一圈仍未找回线后停车。这个“一圈”由命令角速度乘时间估算，不是 IMU 实测角度。

## 速度环审计结论

### 1. 5 ms 单周期计数分辨率是连续性的主要限制

按当前轮周长约 `0.2413 m`、`CPR=728`、控制频率 `200 Hz` 计算：

```text
1 个编码器计数 / 控制周期 ≈ 66.3 mm/s
90 mm/s ≈ 1.36 个计数 / 控制周期
75 mm/s ≈ 1.13 个计数 / 控制周期
```

即使机械匀速，原始速度也会在相邻离散值之间跳变。现有 `alpha=0.4` 低通只能缓和跳变，不能产生真正连续的测速。若实车表现为左右 PWM 高频来回变化，应先检查测量量化与滤波，再盲目修改 `Kp/Ki`。

### 2. 当前 PI 是增量式且存在两个调参相关边界

控制律为：

```text
Pwm += Kp * (error - last_error) + Ki * error
```

- `Pwm` 在 `±7800` 限幅；这限制了最大输出，但不保证目标切换、丢线自转或编码器噪声下无摆动。
- 误差落入 `5 mm/s` 死区时函数直接返回历史 `Pwm`。因此死区并不是把输出归零，而是保持上一输出；恢复运行、目标突变或低速量化时需特别留意这一行为。
- 左右 PI 的内部 `Pwm/Last_bias` 是函数静态变量，目前没有显式“模式切换/丢线/方向翻转时重置”机制。后续若观察到直线转搜线或搜线恢复后出现明显冲击，应优先审查这一状态延续问题。

### 3. 编码器实现会影响 PID 结论

- `Get_Encoder_countA/B` 同时被 GPIO 中断写入和 5 ms 定时器读写，但当前不是 `volatile`。优化编译下不应假定其跨中断可见性绝对可靠。
- 每个端口采用 `if ... else if ...` 只处理一个相位挂起，却会清除两个相位状态；两个相位同时挂起时可能丢计数。

这两项在低速时未必立刻造成问题，但在提速、调 PI 时会把测速噪声误判成控制器问题。修正前应避免对单周期 OLED 数值作过度解读。

## 推荐调参顺序（先测后改）

1. **确认方向**：抬车，中心压线时左右轮均向前；左右偏线时轮速差方向正确。方向不对时禁止调 PI。
2. **隔离速度环**：尽可能让灰度保持中心，使 `Move_Z=0`，观察 `MotorA/B.Target_Encoder`、`Current_Encoder` 与 PWM。先检查两轮是否都能稳定接近相同目标。
3. **先调 Kp**：保持 `Ki` 不变或暂时降低，逐步改变 `Kp`。响应慢才增大；出现周期性快慢交替、PWM 高频摆动时减小。
4. **再调 Ki**：Kp 已能快速跟随后，再只增加足以消除持续速度偏差的 Ki；若直线持续忽快忽慢或两轮反复超调，优先减小 Ki。
5. **最后调巡线外环**：仅在中心直线的速度环稳定后，测试 90/75 mm/s 两档和 `GRAY_STEER_GAIN`。弯道冲出先降低弯道速度或提前降速，不要第一时间增大 PI。

一次实车测试只改变一个参数，记录的是“现象—参数—方向”，不需要复杂性能报表。
